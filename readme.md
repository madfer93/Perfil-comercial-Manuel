## Instalación

1. Clona o descarga el proyecto
2. Abre `index.html` en tu navegador
3. ¡Listo!

## Personalización

### Colores
Edita las variables CSS en `css/style.css`:

```css
:root {
  --primary: #2563eb;
  --secondary: #7e22ce;
  /* ... */
}
```
